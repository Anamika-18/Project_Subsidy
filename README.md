 Income Classification Project (Subsidy Prediction)
 
This project focuses on building a **Binary Classification Model** to predict whether an individual belongs to a **high-income (>50K)** or **low-income (≤50K)** category.
The goal is to assist organizations in making better decisions for subsidy allocation.

---

Objective

* Predict income category based on demographic and financial features
* Perform **Exploratory Data Analysis (EDA)** to identify important patterns
* Reduce unnecessary features while maintaining model performance

---

Work Done (So Far)

* Data Loading & Understanding
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)

* Analysis of **capital-gain** and **capital-loss**
* Relationship between **working hours and income**
* Distribution and categorical feature analysis
* Visualization using **Seaborn & Matplotlib**
